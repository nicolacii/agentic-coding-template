# Skill: Fix Last Task

> Анализ и доработка последней задачи после обнаружения ошибок/недоработок.
> Запуск: `/fix-last-task` или "исправь последнюю задачу", "что-то не так"

---

## Когда вызывать

- AI объявил задачу выполненной, но пользователь нашёл ошибки
- COMPLETION REPORT показал высокую уверенность, но результат неверный
- Challenge / Cross-check были выполнены формально

---

## 7 обязательных фаз

### Phase 1: АНАЛИЗ ПРОВАЛА

```markdown
## ORIGINAL REQUEST
**Что попросил пользователь:** > [точная цитата]
**Что я сделал:** [список действий]
**Что сказал в DONE block:** Уверенность: X%

## ПОВТОРНЫЙ CHALLENGE
1. Как опровергнуть что "задача выполнена"? → [ответ]
2. Открыл ли я ВСЕ файлы и проверил? → [ответ]
3. Какие edge cases пропустил? → [ответ]
4. Решён ли Job пользователя? → [ответ]

## НЕДОРАБОТКИ
| # | Что упущено | Где | Почему пропустил |
|---|-------------|-----|------------------|
```

### Phase 2: ROOT CAUSE (5 Whys)

```markdown
Problem: [что не так]
Why 1: →
Why 2: →
Why 3: →
Why 4: →
Why 5: → [ROOT CAUSE]
```

### Phase 3: ПЛАН ДОРАБОТКИ

Каждая недоработка = отдельный TODO. Начать с критичных.

### Phase 4: ИСПРАВЛЕНИЕ

Для каждого TODO:
1. Пометить in_progress
2. Выполнить исправление
3. Cross-check: открыть файл, прочитать изменения
4. Пометить completed

### Phase 5: ВЕРИФИКАЦИЯ

```markdown
| Артефакт | Открыт | Проверен | Статус |
|----------|--------|----------|--------|

## FINAL CHALLENGE
1-4 вопроса с ответами (все должны быть ДА)
```

### Phase 6: SESSION REVIEW

```markdown
### Где срезал углы
| Момент | Должен был | Сделал вместо | Impact |
|--------|------------|---------------|--------|

### Какой протокол не применил
| Протокол | Статус (❌/⚠️/✅) | Что пропустил |
|----------|-------------------|---------------|
```

### Phase 7: COMPLETION REPORT

```markdown
## FIX LAST TASK — DONE
Найдено недоработок: X
Исправлено: [список]
Root Cause: [почему пропустил]
Финальная уверенность: X%
```

## Запреты

- НЕ пропускать фазы
- НЕ оправдываться вместо анализа
- НЕ объявлять готово без повторного Challenge
