Оценка сложности

Сколько файлов? Какой scope?

🟢

SIMPLE

  • 1-2 файла, typo, стиль
  • Execute → Verify → DONE
  • Без Prompt Prep
  • Без протокола
🟡

STANDARD

  • Фича, баг, несколько файлов
  • Prompt Prep → Plan → Execute → Verify → DONE
  • Routing → Protocol
  • RAT check
🔴

COMPLEX

  • Архитектура, миграция
  • Prompt Prep → Plan → Execute → Verify + CTO Review → DONE
  • Routing → Protocol
  • RAT + Escalation

Verification Flow (ШАГ 3)

4 проверки ▾

📋 Cross-check

Каждый файл ОТКРЫТЬ и проверить ДРУГИМ методом. Таблица: файл → открыт → проверен.

Когда: КАЖДЫЙ ответ

🤔 Challenge (4 вопроса)

1. Как опровергнуть? 2. Все файлы открыл? 3. Edge cases? 4. Job решён?

Когда: КАЖДЫЙ ответ

📊 Confidence Score

100% минус: не тестировал (−50%), не открывал (−40%), нет cross-check (−30%), assumptions (−25%)

Когда: КАЖДЫЙ ответ

📸 Visual Diff

python3 scripts/visual-diff.py {page}
Playwright + pixelmatch → diff image → итерировать пока diff < 1%

Когда: ЛЮБОЕ CSS/SCSS изменение

🎯 RAT (Riskiest Assumption)

Category 0: "Я правильно понял запрос?" Проверить TOP-1 assumption ПЕРЕД кодом.

Когда: STANDARD / COMPLEX

Forbidden Actions

Нарушение = СТОП ▾

❌ Файлы

Нет _fixed, _final, _v2. Нет перезаписи целиком. Нет дубликатов.

❌ Верификация

"Готово" без открытия файлов. "Работает" без запуска. "Совпадает" без diff <1%.

❌ Процесс

Код ДО test cases. Пропуск Prompt Prep. git push без тестов. Пропуск pipeline этапов.

❌ Иконки/Стили

Никогда не рисовать SVG "по памяти". Только из albato-kit/Icon/. Не использовать Tailwind.
