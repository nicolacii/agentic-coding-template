# Implementation Checklist

> Проверить ПЕРЕД началом кодирования (этап 3 pipeline).
> Обновляется из рефлексий — каждый actionable lesson добавляется сюда.

---

## Перед кодом

- [ ] Прочитал `analysis-{section}.md`?
- [ ] Прочитал соответствующий `protocol-*.md`?
- [ ] Выполнил PRE-ACTION (Duplicate Check + JTBD)?
- [ ] Таблица тест-кейсов создана (TDD Phase 0)?
- [ ] Знаю expected output для каждого sub-task?

## Во время кода

- [ ] Один файл за раз?
- [ ] tsc/lint после каждого изменения?
- [ ] Тесты зелёные после каждого шага?
- [ ] Не копирую код — пишу с нуля?

## После кода

- [ ] Task file обновлён [x]?
- [ ] Все файлы открыты и проверены?
- [ ] Visual diff < 1% (если CSS)?
